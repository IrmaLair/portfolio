Identity Bloom

Creative Computing · Oct, 2024 · 1 Week
Exploring one’s identity through art. The project uses one’s name and MBTI personality type to create a personalized, algorithmic visualization. OBJECTIVES Explore the intersection of art and identity Visualize the beauty of individuality through generative art KEY FEATURES Platform: Web browser User input: Participant's name & MBTI Personality type Algorithmic artwork generation Personalized, shareable visual representation
